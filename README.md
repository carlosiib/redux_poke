
# Redux poke

React-redux app with pokeAPI.

## Production web app
1.  [Pokemon-app](https://redux-poke.netlify.app/) using redux and poke API.
2. [Pokemon-app-login](https://poke-redux-login.netlify.app/) with login service by Google.
